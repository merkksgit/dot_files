## Dotfiles

This repository contains my personal configuration files (dotfiles) for various tools and applications. These dotfiles help to maintain a consistent development environment across different machines.

## File locations

### bash

```
~/.bash_aliases
```

```
~/.bashrc
```

### fzf

```
~/.fzf.bash
```

### irssi

```
~/.irssi
```

### lsd

```
~/.config/lsd
```

### nvim

```
~/.config/nvim
```

### tmux

```
~/.tmux.conf
```
